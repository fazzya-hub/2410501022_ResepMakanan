## 
Aplikasi web fullstack berbasis Vue 3 (Composition API) untuk bagian front-end yang bertujuan mendukung digitalisasi UMKM di Indonesia dengan menyediakan platform berbagi resep masakan tradisional secara interaktif.

##
## Fitur Utama
- Menampilkan daftar resep dalam bentuk card (responsive grid)
- Filter resep menggunakan dropdown berdasarkan kategori dan tingkat kesulitan
- Form tambah resep dengan dynamic field (bisa tambah lebih dari 1 bahan dan langkah)
- Validasi form (minimal 1 bahan dan 1 langkah sebelum submit)
- Detail resep menggunakan accordion (collapsible component)
- State management reaktif terhadap perubahan filter dan pagination (Pinia)
- Konfirmasi sebelum submit data
- Feedback setelah aksi (notifikasi sukses/gagal)
- Integrasi API backend Express + MongoDB melalui Axios

## UI/UX
- Menggunakan Bootstrap (opsional pengganti Tailwind CSS)
- Menggunakan component-based design pattern
- Animasi CSS transition saat expand/collapse accordion
- Responsive layout menggunakan grid system
- Modal interaktif untuk tambah resep
- Form tambah resep dirancang user-friendly dengan input dinamis

## Tech Stack
- Vue 3 (Composition API)
- Pinia (State Management)
- Axios
- Bootstrap (UI)
- Express.js (Backend API)
- MongoDB + Mongoose
- Kalau kamu mau, saya bisa buatkan juga versi siap tempel untuk laporan UTS (lebih formal + ada bagian arsitektur dan alur data frontend-backend).
